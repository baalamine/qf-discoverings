\maketitle
 
 Data foresentics with Analytics, shortly DAFNA, is an ambitious project of the data analytics group of Qatar Computing Research
 Institute. It main goal is to provide efficient and effective algorithms and tools which enable user applications to be aware of 
 the veracity level of redundant information from multiple untrustworthy sources they consume. One concrete targeted application
 of this project is to build an end-to-end truth discovery systems that provides reliable information about facts in Qatar, given 
 some extraction and integration phases. This challenge asks for dealing with diverses research sub-problems related to fact extraction,
 studys of existing truth finding algorithms, and inferrence of an efficient approach.

 
 
 We will present our ongoing study to compare existing truth discovery algorithms, providing an API for using those algorithms, and finally
 the coupling of the truth finding process with real information extraction systems.
